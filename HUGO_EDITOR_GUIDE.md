# Hugo 集成编辑器使用指南

## 功能概述

PineappleBlog Editor 现已完全集成 Hugo 文章结构，提供专业的博客文章编辑体验。

## 主要功能

### 1. **Front Matter 编辑器**
Front Matter 是 Hugo 文章的元数据部分，包含文章的各种配置信息。

#### 支持的字段：
- **标题 (title)**: 文章标题
- **发布日期 (date)**: 文章发布时间，使用日期时间选择器
- **草稿状态 (draft)**: 标记文章是否为草稿
- **描述 (description)**: 文章简短描述，用于 SEO 和摘要
- **标签 (tags)**: 文章标签，支持多个标签
- **分类 (categories)**: 文章分类，支持多个分类

#### 操作方式：
- 直接编辑各个字段
- 标签和分类：输入后按回车或点击"添加"按钮
- 点击标签/分类旁的 × 删除
- 点击"收起/展开"按钮折叠 Front Matter 编辑器

### 2. **Markdown 工具栏**
快速插入常用的 Markdown 格式：

| 工具 | 图标 | 功能 | 快捷键 |
|------|------|------|--------|
| 粗体 | 𝐁 | 加粗文本 | Ctrl+B |
| 斜体 | 𝐼 | 斜体文本 | Ctrl+I |
| 标题 | H | 插入标题 | - |
| 链接 | 🔗 | 插入链接 | - |
| 图片 | 🖼️ | 插入图片 | - |
| 代码 | </\> | 插入代码块 | - |
| 引用 | ❝ | 插入引用 | - |
| 列表 | ≡ | 无序列表 | - |
| 有序 | 1. | 有序列表 | - |
| 任务 | ☑ | 任务列表 | - |
| 表格 | ⊞ | 插入表格 | - |
| 分割线 | ― | 插入分割线 | - |

### 3. **实时预览**
- 左侧编辑，右侧实时预览
- 支持 GitHub Flavored Markdown
- 代码语法高亮
- 300ms 防抖优化性能
- 点击"显示/隐藏预览"切换预览面板

### 4. **自动保存**
- 编辑停止 3 秒后自动保存
- 顶部显示保存状态（未保存/已保存）
- 支持 Ctrl/Cmd+S 手动保存

### 5. **统计信息**
底部状态栏显示：
- 行数
- 字数
- 字符数
- 预计阅读时间

## Hugo 文章结构

编辑器自动处理 Hugo 文章的完整结构：

```markdown
---
title: "文章标题"
date: 2025-01-18T15:30:00+08:00
draft: false
description: "文章描述"
tags:
  - 标签1
  - 标签2
categories:
  - 分类1
---

# 文章内容

这里是 Markdown 正文...
```

## 工作流程

### 创建新文章
1. 在项目视图点击"✍️ 新建文章"
2. Front Matter 自动生成默认值
3. 编辑标题、标签、分类等元数据
4. 在 Markdown 编辑区写作内容
5. 实时预览效果
6. 自动保存或手动保存

### 编辑现有文章
1. 在文章列表点击文章
2. 编辑器自动解析 Front Matter
3. 所有元数据显示在顶部编辑器
4. Markdown 内容显示在下方编辑器
5. 修改后自动保存

## 技巧与注意事项

### Front Matter 最佳实践
- **标题**: 简洁明了，包含关键词
- **日期**: 保持准确，影响文章排序
- **草稿**: 未完成的文章标记为草稿
- **描述**: 100-160 字符，用于搜索引擎
- **标签**: 3-5 个相关标签
- **分类**: 1-2 个主要分类

### Markdown 编辑技巧
- 使用工具栏快速插入格式
- 支持所有标准 Markdown 语法
- 支持 GFM（表格、任务列表、删除线等）
- 代码块支持语法高亮（在预览中）

### 性能优化
- Front Matter 可折叠以节省空间
- 预览可隐藏以专注写作
- 300ms 防抖避免频繁渲染

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl/Cmd+S | 保存文章 |
| Ctrl+B | 粗体（计划中） |
| Ctrl+I | 斜体（计划中） |
